# Enterprise Healthcare Analytics Dashboard

A Power BI healthcare analytics project built to turn raw hospital data into executive-ready insights.

## Overview

This dashboard helps healthcare leaders track patients, billing, insurance coverage, medical conditions, and diagnostic results in one place.

## What this project solves

Healthcare teams often deal with scattered data and slow reporting. This project centralizes the analysis so stakeholders can monitor performance, spot risk, and make faster decisions.

## Key business goals

- Improve visibility into patient volume and demographics
- Track billing and revenue patterns
- Compare insurance provider performance
- Monitor medical condition distribution
- Review diagnostic outcomes and abnormal cases
- Support executive decision-making with clear KPIs

## Dashboard pages

### 1. Medical Condition Analysis
Focuses on patient distribution by condition, hospital, and geography.

### 2. Insurance Distribution Analysis
Shows how insurance providers are distributed and how they affect billing.

### 3. Test Result Analysis
Highlights normal versus abnormal reports and helps identify higher-risk areas.

### 4. Executive KPI Dashboard
Summarizes the most important business metrics in one view.

## Core KPIs

- Total Patients
- Total Billing
- Average Billing
- Total Admissions
- Average Length of Stay
- Average Age

## Data model

The project follows a star schema design to keep the model scalable, improve performance, and simplify reporting.

## Tools used

- Power BI
- Power Query
- DAX
- Excel
- Star Schema Modeling

## Insights

- Some medical conditions drive a larger share of admissions and billing.
- Insurance coverage is a major factor in financial analysis.
- Most test results are normal, but abnormal cases need closer review.
- Patient distribution and billing vary across hospitals, which supports operational comparison.

## Business recommendations

- Prioritize high-volume conditions for staffing and resource planning
- Use abnormal test patterns to flag potential operational or clinical risk
- Review hospital-level billing differences to improve financial strategy
- Monitor insurance mix to understand revenue exposure
- Use executive KPIs for faster leadership reporting

## Why this project matters

This dashboard is not just a visual report. It is a decision-support tool that combines ETL, modeling, KPI design, and storytelling to deliver practical healthcare intelligence.

## Final takeaway

The project demonstrates how raw healthcare data can be transformed into a clean, interactive, and business-focused dashboard that supports both operational monitoring and executive planning.
