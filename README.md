# 🏥 Enterprise Healthcare Analytics Dashboard

A comprehensive healthcare analytics solution designed to transform raw patient and hospital data into actionable insights.

Built using **Power BI**, **Power Query**, **Excel**, and **DAX**, this dashboard provides interactive visualizations to monitor patient distribution, billing performance, hospital operations, and treatment outcomes.
