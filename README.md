# 🧮 DAX Highlights

```DAX
Total Patients

DISTINCTCOUNT(Patient[Patient_ID])
```

```DAX
Total Revenue

SUM(Finance[Revenue])
```

```DAX
Average Stay

AVERAGE(Hospital[Length_of_Stay])
```

```DAX
Readmission Rate

DIVIDE(
    [Readmitted Patients],
    [Total Patients]
)
```

```DAX
Bed Occupancy %

DIVIDE(
    [Occupied Beds],
    [Total Beds]
)
```

# 🚀 Getting Started

> [!NOTE]
> Follow these steps to explore the Enterprise Healthcare Analytics Dashboard locally.

## Prerequisites

- ✅ Power BI Desktop (Latest Version)
- ✅ Windows 10/11
- ✅ Basic understanding of Power BI

---

## Installation

```bash
# Clone Repository

git clone https://github.com/yourusername/Enterprise-Healthcare-Analytics-Dashboard.git

cd Enterprise-Healthcare-Analytics-Dashboard
```

Open:

```
Dashboard/
└── Enterprise Healthcare.pbix
```

Refresh the dataset if required.
